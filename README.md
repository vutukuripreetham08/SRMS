# SRMS â€“ Student Result Management System

SRMS is a C-based command-line program to manage student records and exam scores. It allows users to add, view and edit student data stored in a simple text-based format. The repository includes source files (`SRMS_AP24110010605.c`, `Simulation_SRMS.c`), a sample student list (`students.txt`), and configuration (`Credentials.txt`).  

## Features  
- Add new student records  
- Update existing student information  
- Display exam results for any student  

## How to build  
Compile all `.c` files using a C compiler (e.g., `gcc`) and run the resulting executable.
